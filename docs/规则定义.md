# 规则定义

本仓库只定义 L0 内核可识别的规则文件格式和加载方式，不提供任何业务侧规则集。

## 内核支持的规则文件

- `RepoPolicy.yaml`
  - 定义规则版本、分片亲和性、资源限制、基础合规规则。
- `ToolPolicy.yaml`
  - 定义工具白名单、黑名单、权限范围。
- `ContractSpec.json`
  - 定义统一输入契约与统一输出契约。
- `StyleGuide.md`
  - 定义内核或业务仓库需要遵循的编码与评审规则。
- `TestTargets.yaml`
  - 定义单测、集成、契约校验命令。

## 仓库内的示例规则

仓库只保留纯内核示例规则，位于：

- `docs/examples/minimal-specs`

这组示例仅用于展示规则格式和验证工具用法，不代表任何业务实现。
